Jewel Flickr Plugin is a Simple Flickr Plugin to show Flickr Photos in your sidebar.

== Description ==

### Show your flickr photos form flickr, in an easy and fun way! ### 

= Features: =

* <strong>Displays Flickr photos as the style you want.</strong> 
* <strong>You can select the images in a number of coloumns that you want.</strong> 
* <strong>Fits into a sidebar widget easily.</strong>
* <strong>Choose display type (vertical or horizontal), define no. of columns.</strong>    
* <strong>Make your own layout.</strong>
* <strong>Order the icons - they way you like by using easy drag and drop feature.</strong> 
* <strong>Fully Responsive - compatible with mobile devices</strong>
* <strong>User friendly and very interactive user interface</strong 
* <strong>Upgrades available</strong> 
    - For more features like flickr image slider, more customization options - upgrade to Pro. 
* <strong>Support</strong> 
    - Dedicated email, forum support 
* <strong>Free updates</strong>
    - Get free updates for lifetime.
